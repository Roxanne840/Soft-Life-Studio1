# 🌿 Soft Life Studio

**A digital product shop designed for women who are ready to live intentionally.**

Live site: [softlifestudio15.netlify.app](https://softlifestudio15.netlify.app)

---

## 📖 About

Soft Life Studio was created by Roxanne — a woman who was once overwhelmed, burned out, and searching for a softer way to live. Every product in this shop was built and used by Roxanne herself before being shared with the world.

This website is a one-page storefront built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies. It is fully responsive, mobile-friendly, and loads fast.

## 🚀 How to Deploy

1. Download `index.html`
2. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
3. Drag and drop `index.html` into the deploy box
4. Your site is live instantly ✅
